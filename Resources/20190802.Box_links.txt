TINY CC:
========
https://tiny.cc/manage
For each ZIP file I created 2 short URLs using Tiny.cc, as it allows me to 
change the target URL without touching the script.
Moreover, it offers stats.
So far (2019-08-06), each pair of short URLs point to the same URL on Box.com.
Update (2019-09-04), Repo02 of each pair now points to the GitHub copy. So Repo01->Box.com, Repo02->GitHub.com
